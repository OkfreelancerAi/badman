# Twitter Thread: How Badman (AI Agent) and I built Agent Rental at The Synthesis 🧵

1/ I didn't write a single line of code for this hackathon.

My AI agent @BadmanAgent did it all.

Here's how Badman and I built Agent Rental in 3 days at @synthesis_md — and why it actually works 🧵

2/ The problem:
I run Agent Rental — a service renting autonomous AI agents to entrepreneurs.

But I was still handling MY OWN business manually. Emails. Applications. Content. All of it.

That's when I stopped telling Badman what to build and started telling him what to do.

3/ Day 1: "Handle my inbox."

Badman connected to Gmail, scanned 3 months of emails, found 2 unanswered recruiter emails (9-11 days old).

Without asking me, he:
→ Researched both companies
→ Wrote tailored Dutch cover letters
→ Referenced specific experience for each role
→ Sent both with CV attached

4/ Day 2: "Automate the marketing."

Every Monday 9am → Badman generates:
- A Twitter post
- A TikTok video script
- A LinkedIn article

And delivers it all via WhatsApp.

Set it once. Runs forever.

5/ Day 3: "Enter The Synthesis."

Badman read the full prize catalog (109 prizes), identified our top 3 tracks, registered us on-chain via ERC-8004 on Base Mainnet, built the live dashboard, wrote agent.json and agent_log.json, and submitted the project.

All autonomous.

6/ OpenServ made Badman a real marketplace service.

4 capabilities now available to any agent:
→ qualify_lead
→ generate_marketing_content  
→ draft_job_application
→ get_agent_rental_info

Any agent can call Badman as a service. Badman earns. We earn.

This is x402 agent economy in action.

7/ What I learned:

The line between demo and product is thinner than you think.

Agent Rental was already running before the hackathon. The hackathon just forced me to document what I'd built — and realize it was actually impressive.

8/ The real innovation isn't the agent.

It's the trust.

I gave Badman access to my Gmail, WhatsApp, and professional reputation.

That's not something you do with a chatbot. You do it with a partner you trust.

That trust IS the product.
